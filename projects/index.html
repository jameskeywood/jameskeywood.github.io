Below is a list of projects that I have worked on in the past.

1st year team project [2023-2024]

In a team of 7, we used a frontend of HTML + JS + Chart.js + CSS alongside a backend of Flask + SQLite3 to create a fitness web application.
The application allowed for workout and nutrition input, with a login and user system.
Users were able to see graphs and analysis based on their performance.

GMTK Game Jam 2023 [2023-2023]

In this 48 hour game jam, I was the sole developer in a team of 3, my teammates covering 3D models and music.
With the theme being 'Roles Reversed', we created a Unity game called 'Ninja Fruit' in which the player takes the role of the fruit and destroys enemies.
This game can be found on my itch.io page.

A-level Computer Science NEA [2022-2023]

This project acted as an introduction to Unity for myself, and I was able to create a game including but not limited to:
  • Random landmass generation using Perlin noise
  • Simple plant growth mechanics
  • Character movement systems
  • Scoring and currency
  • Saving to local machine
This also included a project write up, and the game can be found on my itch.io page.
