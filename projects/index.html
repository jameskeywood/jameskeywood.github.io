Starmapper [2024-2024]

Manchester University Student Hackathon 2024, with a theme of 'cosmic'.
Within this 24 hour period, myself and two team members created a web application to process images of the night sky.
One function counts and highlights any stars or clusters in the image.
The other uses a trained machine learning model to inform the user which constellation, if any, is in their image.

lightr [2023-2024]

For my first year team project, the 7 of us used a frontend of HTML + JS + Chart.js + CSS alongside a backend of Flask + SQLite3 to create a fitness web application.
The application allowed for workout and nutrition input, with a login and user system.
Users were able to see graphs and analysis based on their performance.

A-level Computer Science NEA [2022-2023]

For my A-level Computer Science NEA, I made a Unity game focused on gardening. This project acted as an introduction to Unity, and I was able to implement features including but not limited to:
  • Random landmass generation using Perlin noise
  • Simple plant growth mechanics
  • Character movement systems
  • Scoring and currency
  • Saving to local machine
This also included a project write up, and the game can be found on my itch.io page.
